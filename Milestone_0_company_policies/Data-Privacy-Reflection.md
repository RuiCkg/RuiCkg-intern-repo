# Data Privacy & Confidentiality - Rui Chosa

## Research & Learn
### What types of data are considered confidential at Focus Bear?
- Identification Data
- Socail Data
- Academic & Professional Data
- Commercial and Marketing Data
- Technical Data
- Economic, Financial and Insurance Data
- Profile Data
- Aggregate Data
- Special Category of Personal Data
### What are best practices for handling confidential data?
- Only access or use confidential data for work purposes that require it.
- Store sensitive information in secure, official systems rather than personal devices.
### How should you respond to a suspected data breach or accidental disclosure of confidential information?
- Report the breach
- Don't share the confidential data

## Reflection
### What steps can you take to ensure you handle data securely in your daily tasks?
- Secure access: strong password, multi-factor authentication
- Device Saftey: lock your screen, keep software updated, encrypt data
- Daily habits: Back up data regularly, report suspicious activity

### How should you store, share, and dispose of sensitive information safely?
- Using encryption and password protection, store in secure places.
- Using secure file sharing services and verify who is asking for the information.

### What are some common mistakes that lead to data privacy issues, and how can they be avoided?
- Sharing confidential data to anyone in the world
- Storing data in personal locations
