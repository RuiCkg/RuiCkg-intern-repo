# VS Code Shortcuts - Rui Chosa

## Navigation
- **Ctrl + P:** Quickly open files by name  
- **Ctrl + Shift + O:** Navigate to symbols (functions, methods) in a file  
- **Ctrl + Tab:** Switch between open files  

## Editing
- **Alt + Click:** Multiple cursors (edit multiple lines at once)  
- **Ctrl + D:** Select the next occurrence of the current word  
- **Ctrl + /:** Toggle line comment  
- **Alt + Up / Down:** Move the current line up or down  

## Debugging
- **F5:** Start debugging  
- **F9:** Toggle breakpoint  

## Terminal Commands
- **Ctrl + `** Open or close the integrated terminal  


## Most Useful Shortcuts for Daily Work
Ctrl + ` will be useful since we need terminal a lot for developing without clicking terminal button everytime when you need.

## How Keyboard Shortcuts Improve Productivity
It will reduce a lot of time instead using mouse everytime when you need to do something while you develop.
